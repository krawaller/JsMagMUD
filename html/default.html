Welcome to JsMagMUD

JsMagMUD is a community project for the members of, but not exclusive to, JsMag. The purpose is to implement a fully functional MUD server and client using nothing more than JavaScript, HTML, and CSS.

The server portion of JsMagMUD is 100% JavaScript running on a Node.js server

The client portion of JsMagMUD is HTML, CSS, and JavaScript. The only exception is where a Flash compontent is used when WebSockets, or another better alternative to Flash, is not available for communications.
